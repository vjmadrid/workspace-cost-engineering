## Configuración

### Crear una cuenta de Anthropic Console

Nota: Si ya se tiene una cuenta disponible, se puede omitir esta sección

Pasos a seguir:

* Acceder a la URL: [https://console.anthropic.com](https://console.anthropic.com)
* Crear una cuenta nueva pulsando en "Sign Up"
  * Opción 1: Crear una cuenta desde el email
  * Opción 2: Crear una cuenta desde Google
* Completar el formulario de registro con los datos solicitados
* Verificar la cuenta a través del correo electrónico recibido
* Verificar que se ha creado por SMS, etc.

En ocasiones las cuentas nuevas disponen de una serie de créditos gratuitos para probar la plataforma, por lo que se recomienda revisar el apartado de créditos disponibles para verificar si se dispone de ellos y así poder empezar a probar la plataforma sin necesidad de configurar la facturación inicialmente.

### Configuración de la facturación

Pasos a seguir:

* Acceder a la Plataforma de Anthropic
* Acceder a la opción de "Settings" -> "Billing"
* Añadir un mínimo de creaditos: entre $5 y $10 para empezar a probar la plataforma
* Acceder a la opción de "Add credit"
* Realizar el pago con la tarjeta de crédito o débito
* Revisar que se ha cargado adecuadamente el dinero
* Opcional: Configurar alertas de gasto para evitar sorpresas en la factura etc (acceder a "Usage limits" para configurar límites de gasto, alertas, etc)

Nota: Funciona con un sistema de prepago

### Generar la API Key

Pasos a seguir:

* Acceder a la Plataforma de Anthropic
* Acceder a la opción "API Keys" del menu lateral
* Acceder a la opción "Create key"
* Seleccionar un nombre para la API Key descriptivo y adecuado
* Copiar la key inmediatamente, ya que no se podrá volver a visualizar posteriormente por seguridad

**Importante**: Hay que tener mucho cuidado con quien se comparte la API Key, ya que es la llave de acceso a la plataforma y a los recursos asociados a la cuenta, incluyendo los créditos disponibles. Compartir la API Key con personas no autorizadas puede resultar en un uso indebido de la plataforma y en cargos inesperados en la factura. Se recomienda almacenar la API Key de forma segura y no compartirla públicamente ni con personas no autorizadas. En caso de sospechar que la API Key ha sido comprometida, se debe revocar inmediatamente y generar una nueva para garantizar la seguridad de la cuenta.
