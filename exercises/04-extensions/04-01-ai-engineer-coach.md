# 04-01 ai engineer coach

## Objetivo

El ejercicio esta pensado para ayudar a entender la extensión de "AI Engineer Coach" y cómo puede ser utilizada para mejorar la productividad y eficiencia en el desarrollo de proyectos de inteligencia artificial. En este ejercicio se explicará qué es esta extensión, cómo se puede utilizar y cuáles son sus beneficios.

## Requisitos previos

* Visual Studio Code instalado en el equipo
* Extensión de "AI Engineer Coach" instalada en Visual Studio Code

## Estimación de tiempo

Tiempo estimado en complentarse: 5-10 minutos

## Instrucciones

### Paso 1: Cargar la extensión de "AI Engineer Coach" en Visual Studio Code

Seleccionar el arrancar la extensión desde el menú de la izquierda de Visual Studio Code o bien desde la paleta de comandos y buscar "AI Engineer Coach: Open Dashboard" para arrancar la extensión

### Paso 2: Sincronizar con los datos locales

Sincronizar los datos de las sesiones locales para que se puedan utilizar en la extensión y así poder obtener recomendaciones personalizadas basadas en el historial de sesiones y proyectos del usuario

### Paso 3: Revisar las recomendaciones y consejos proporcionados por la extensión

Revisar las recomendaciones y consejos proporcionados por la extensión en función de los datos sincronizados y cómo pueden ayudar a mejorar la productividad y eficiencia en el desarrollo de proyectos con inteligencia artificial
