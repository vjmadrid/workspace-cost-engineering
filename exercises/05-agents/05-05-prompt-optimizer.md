## Instrucciones

### Paso 1: Verificar que se encuentra activo el agente

Verficar que el agente se encuentra cargado y activo en alguno de los ambientes de trabajo mostrado (github copilot, chat IA conversacional o modo agente IA conversacional)

### Paso 2: Probar diferentes ejemplos de prompts de usuario

Algunos ejemplos de uso pueden ser los siguientes:

Nota: Recordar que se le puede "apretar" con el prompt o bien cambiar un poco el formato de respuesta

```bash
# ***Nivel 1 - Limpieza básica (redundancia y claridad)***
Quiero que me expliques de forma clara, sencilla, fácil de entender, con ejemplos y bien explicado qué es Kubernetes y cómo funciona, intentando que sea lo más completo posible pero también resumido

Necesito que analices este texto y me hagas un resumen, pero que sea un resumen bien hecho, completo pero no muy largo, y que incluya las cosas importantes y las ideas clave

# ***Nivel 2 - Optimizacion estructural (orden y delimitacion)***
Genera ideas de negocio innovadoras, nuevas, originales, creativas y diferentes que no se hayan visto antes.

Tengo este documento de arquitectura y necesito ver si es correcto, detectar problemas, sugerir mejoras y además explicar todo con detalle para que cualquiera pueda entenderlo.

# ***Nivel 3 - Optimización de costes***
Analiza este repositorio completo y genera documentación detallada de todos los módulos con ejemplos

Te paso varios documentos y quiero que me hagas un informe completo con todo lo relevante, análisis, conclusiones y recomendaciones

# ***Nivel 4 - Production-ready***
Genera una API en Python con FastAPI que reciba datos, los procese y devuelva resultados. Quiero que el código esté bien estructurado, documentado y explicado

# AYUDA EXTRA



```
